### Vampire Survivors Lite  V1.0

> 网页版原型：**移动躲怪 + 自动攻击 + 掉落经验 + 升级三选一 + 多武器叠加 + 敌人随时间变强 + 计时生存**。

**“能玩”的 Survivors 核心原型**

它包含：

- ✅ 无尽地图（相机跟随、背景网格）
- ✅ 敌人自动追踪玩家，随时间变强/变多
- ✅ 自动攻击（飞刀/追踪弹），多投射物/穿透
- ✅ 近身光环（蒜圈）
- ✅ 环绕武器（符文环绕）
- ✅ 经验宝石拾取、升级暂停、三选一成长
- ✅ 基础被动（伤害/冷却/范围/移速/护甲/回血/吸取/弹速）
- ✅ 手机虚拟摇杆 + 冲刺/暂停按钮
- ✅ 本地存档：记录最佳存活时间（localStorage）
