### Vampire Survivors Lite  V2.2

1. 难度系统（三档）
   - 菜单新增 爽爆 / 中等 / 困难 按钮，支持保存到 localStorage。
   - 难度会联动：XP 曲线、连锁窗口/奖励、怒气获取/持续、暴击基准、敌人成长、刷怪速率、初始武器。
2. 连锁击杀（Combo）
   - 新增 comboCount/comboTimer/comboMultiplier。
   - 规则：击杀刷新窗口，超时归零。
   - 奖励：每 5 连触发小爆炸+额外 XP；每 15 连触发大爆炸+大 XP+临时暴击加成。
   - XP 掉落按连锁倍率放大。
3. 怒气系统（Rage）
   - 新增 rageValue、击杀/受伤增怒、时间衰减。
   - 满 100 自动进入 Rage Mode（有持续时间），结束归零。
   - Rage 效果：伤害、攻速、移速提升 + 轻微震屏。
4. 暴击系统（Crit）
   - 每次伤害判定暴击，暴击伤害倍率生效。
   - 暴击率上限强制 40%。
   - 暴击成长路径：新增被动 暴击率 / 暴击伤害，并与 Rage、Combo 临时 buff 联动。
5. 机制联动与爽感增强

- HUD 新增 连锁/怒气/暴击 实时显示。
- 新增伤害飘字、暴击高亮、连锁爆发特效、Rage 震屏。

### Vampire Survivors Lite  V1.0

> 网页版原型：**移动躲怪 + 自动攻击 + 掉落经验 + 升级三选一 + 多武器叠加 + 敌人随时间变强 + 计时生存**。

**“能玩”的 Survivors 核心原型**

它包含：

- ✅ 无尽地图（相机跟随、背景网格）
- ✅ 敌人自动追踪玩家，随时间变强/变多
- ✅ 自动攻击（飞刀/追踪弹），多投射物/穿透
- ✅ 近身光环（蒜圈）
- ✅ 环绕武器（符文环绕）
- ✅ 经验宝石拾取、升级暂停、三选一成长
- ✅ 基础被动（伤害/冷却/范围/移速/护甲/回血/吸取/弹速）
- ✅ 手机虚拟摇杆 + 冲刺/暂停按钮
- ✅ 本地存档：记录最佳存活时间（localStorage）
